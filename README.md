## Installation and start

In order to execute host_microbe_PPI you need to install following R packages:
library(d3heatmap)
library(gplots)
library(scales)
library(shiny)
library(shinyalert)
library(shinyBS)
library(shinyjs)
library(shinythemes)
library(xlsx)

If the packages are installed, it can be executed via

library("ui.R)
library("source.R")
shinyApp(ui,server)
